# SKYWAY KOP – Online Flight Booking System

## 📌 Project Overview
A web-based flight booking system developed as a BCA final year project.

## 🚀 Features
- Flight search
- Seat selection
- Booking system
- Boarding pass generation

## 🛠️ Technologies Used
- HTML5, CSS3, JavaScript
- PHP, MySQL

## 📂 Project Structure
- Frontend: HTML, CSS, JS
- Backend: PHP
- Database: MySQL
